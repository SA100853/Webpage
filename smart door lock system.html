Smart Door Lock System

This project uses Arduino uno, RFID sensor, ultrasonic sensor, display LCD1602, I2C module, green LED, Red LED, Buzzer, breadboard and connecting wires

the algorithm of this working model is arduino uno connecting with lcd display 1602A with i2c modul, ultrasonic sensor, servomotor, RFID-RC522, green LED, Red Led , Buzzer when any object comes in range of one meter of ultrasonic sensor LCD1602 Display shows Verify your ID and then RFID verify its ID using RFID tag then after verification servomotor ratates resulting opening the Door and buzzer sounds beep beep and red led Blinks and when on one comes in range green led glows LCD1602 display shows authentication required
