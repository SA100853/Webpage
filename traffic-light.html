Autonomous traffic-light Managment System

This project uses Arduino uno, ESP-01(WiFi-moduel), PIR sensor, display LCD1602, I2C module, green LED, Red LED, Buzzer, breadboard and connecting wires

the algorithm of this working model is when there is heavy traffics on roads PIR sensor along with other sensors senses the thermal radiations and gives this data wirelessly to traffic control office through ESP-01 and they handle traffic lights accordingly
