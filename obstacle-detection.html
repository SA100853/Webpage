Obstacle Detection System

This project uses Arduino uno, ultrasonic sensor, display LCD1602, I2C module, green LED, Red LED, Buzzer, breadboard and connecting wires

the algorithm of this working model is arduino uno connecting with lcd display 1602A with i2c modul, ultrasonic sensor, green LED, Red Led and Buzzer when any object comes in range of two meter of ultrasonic sensor Display shows object detected! and then red LED blinks and buzzer sounds beep-beep

and when no any object comes in range then display shows no object found! and green LED glows
