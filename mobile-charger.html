Mobile-Charger

This project uses step-down transformer(12v-0v-12v), full wave rectifier, 1k ohm resistor, 10mF capacitor, Multimeter, breadboard and connecting wires

the algorithm of this working model is we give input supply of ~240V AC current to the step-down transformer now current is converted into ~12v AC then current passes through Full wave rectifier here it converted into DC current then through resistors and capacitors and finally output will ~5v DC and ~2Amp when we measure through multimeter
